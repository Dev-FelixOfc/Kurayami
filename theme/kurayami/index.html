Guía de Instalación Rápida

  1. Accede a tu servidor mediante la terminal e ingresa al directorio de temas:
    cd ~/KazumaApi/themes
  2. Clona el repositorio oficial del tema ejecutando:
    git clone https://github.com/Dev-FelixOfc/Kurayami.git kurayami
  3. Entra a tu **Panel de Administración** en la web de la API.
  4. Activa el tema seleccionando Kurayami y guarda los cambios para aplicarlo globalmente.
